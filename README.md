
# BookStore

Full stack web application that enable users to view and search books and admin to list books

## Technology Stack

### Back-End 

- Express JS
- Node JS

 Libraries Used

 - Resend
 - Cloudinary
 - JWT

### Front-End

- React JS
- JavaScript
- Tailwind CSS

### Database

MongoDB

### Containerization

Docker
 

 





## Features

- User login
- Use register
- Admin can add and edit books
- Users can search and view books
